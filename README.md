# VK Mini App: Оптимальный линейный раскрой

Мини-приложение ВКонтакте на `JavaScript + React + VKUI + VK Bridge` для расчёта оптимального линейного раскроя по эвристике `FFD (First Fit Decreasing)`.

## Возможности

- проект с несколькими разными заготовками;
- пользовательские шаблоны заготовок;
- ввод деталей с очисткой списка;
- расчёт раскроя с учётом длины заготовки, пропила и стоимости;
- история раскроев через `VKWebAppStorageGet` / `VKWebAppStorageSet` с fallback на `localStorage`;
- графическая схема раскроя;
- текстовая схема раскроя;
- размеры от нуля;
- интерактивное перемещение и масштабирование схемы:
  - мышью;
  - тачпадом;
  - touch / pinch на смартфоне;
- экспорт:
  - текстовый отчёт (TXT или копирование в буфер в мобильном приложении);
  - PDF с графическими схемами и остатком по каждой заготовке (на десктопе / в браузере);
- сообщения в статус-баре автоматически сбрасываются через несколько секунд;
- поддержка собственных единиц длины и обозначения валюты.

## Используемые VK API

- `VKWebAppInit`
- `VKWebAppGetUserInfo`
- `VKWebAppStorageGet`
- `VKWebAppStorageSet`
- `VKWebAppSetSwipeSettings`

## Экспорт

Опция «Включать размеры от нуля» действует и для TXT, и для PDF.

### В браузере (vk.ru, m.vk.ru)

- **Скачать TXT** — скачивается файл с текстовой схемой раскроя.
- **Скачать PDF** — скачивается PDF с графическими схемами (включая блок «Остаток» по каждой заготовке), текстовой сводкой и при необходимости размерами от нуля. Доступен выбор ориентации: вертикально или горизонтально.

### В мобильном приложении VK

- Кнопка **«Скопировать данные расчёта»** — текстовая схема копируется в буфер. Можно вставить в заметки или чат.
- Кнопки скачивания PDF и выбора ориентации в приложении скрыты. Отображается примечание: *«Расчёт в PDF можно скачать только через веб-браузер»* (открыть мини-приложение в браузере или с компьютера).

## Размещение на VK Hosting

Проект подготовлен под официальный хостинг мини-приложений ВКонтакте.

### 1. Укажите `app_id`

В файле `vk-hosting-config.json` укажите свой `app_id` приложения из [кабинета разработчика VK](https://dev.vk.com/).

### 2. Установите зависимости

```bash
npm install
```

### 3. Выполните деплой

```bash
npm run deploy
```

Для этого используется `@vkontakte/vk-miniapps-deploy`.

Важно:

- у проекта Vite, поэтому в `vk-hosting-config.json` используется `static_path: "dist"`;
- после деплоя пакет может предложить автоматически обновить URL в настройках приложения VK.
