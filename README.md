# VK Mini App: Оптимальный линейный раскрой

Мини-приложение ВКонтакте на `JavaScript + React + VKUI + VK Bridge` для расчёта оптимального линейного раскроя по эвристике `FFD (First Fit Decreasing)`.

## Возможности

- проект с несколькими разными заготовками;
- пользовательские шаблоны заготовок;
- ввод деталей с очисткой списка;
- расчёт раскроя с учётом длины заготовки, пропила и стоимости;
- история раскроев через `VKWebAppStorageGet` / `VKWebAppStorageSet` с fallback на `localStorage`;
- графическая схема раскроя;
- текстовая схема раскроя;
- размеры от нуля;
- интерактивное перемещение и масштабирование схемы:
  - мышью;
  - тачпадом;
  - touch / pinch на смартфоне;
- экспорт:
  - `TXT`;
  - `PDF`;
- поддержка собственных единиц длины и обозначения валюты.

## Используемые VK API

- `VKWebAppInit`
- `VKWebAppGetUserInfo`
- `VKWebAppStorageGet`
- `VKWebAppStorageSet`
- `VKWebAppSetSwipeSettings`

## Экспорт

### TXT

- выгружает текстовую схему раскроя;
- можно включать или выключать размеры от нуля.

### PDF

- выгружает графические схемы раскроя;
- добавляет текстовую схему;
- можно включать или выключать размеры от нуля;
- можно выбирать ориентацию листа: вертикально или горизонтально.

## Размещение на VK Hosting

Проект подготовлен под официальный хостинг мини-приложений ВКонтакте.

### 1. Укажите `app_id`

В файле `vk-hosting-config.json` нужно заменить:

```json
"app_id": 0
```

на реальный идентификатор приложения из кабинета разработчика VK.

### 2. Установите зависимости

```bash
npm install
```

### 3. Выполните деплой

```bash
npm run deploy
```

Для этого используется `@vkontakte/vk-miniapps-deploy`.

Важно:

- у проекта Vite, поэтому в `vk-hosting-config.json` используется `static_path: "dist"`;
- после деплоя пакет может предложить автоматически обновить URL в настройках приложения VK.
